# AssetFlow ERP

Enterprise Asset & Resource Management System built using React.

## Features

- Asset Management
- Booking Management
- Maintenance Tracking
- Inventory Management
- Procurement
- Vendor Management
- Reports & Analytics
- User Management
- Notifications
- Settings

## Tech Stack

- React
- Material UI
- React Router
- Axios
- Recharts

## Running the Project

### Option 1: HTML Demo (Recommended for Evaluation)

The HTML demo cannot be executed using a normal Python compiler or by simply opening the file in some environments.

To run it:

1. Install the **Live Server** extension in **Visual Studio Code**.
2. Open the project folder in VS Code.
3. Right-click the `AssetFlow.html` file.
4. Select **"Open with Live Server"** (or **"Open on Live Server"**, depending on your VS Code version).

The application will automatically open in your default browser.

### Option 2: React Project

```bash
npm install
npm start
```

or

```bash
npm install
npm run dev
```

(depending on the project setup)
